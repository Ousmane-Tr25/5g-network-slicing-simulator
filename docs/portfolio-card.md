# Portfolio card

## 5G Network Slicing Simulator

Python simulation project for allocating synthetic 5G users to network slices under different traffic conditions.

**Main goal:** compare latency-threshold, K-Means and random allocation methods using slice load, capacity utilization, overload detection and balance metrics.

**Tech stack:** Python, NumPy, Pandas, Scikit-learn, Matplotlib, Pytest, Docker, GitHub Actions.

**ML methods:** K-Means clustering, Linear Regression.

**Why it is useful for recruiters:** the project converts a master research topic into a reproducible, documented and tested ML simulation project.

## Suggested GitHub description

`5G network slicing simulator with K-Means traffic clustering, resource utilization metrics, reproducible charts, tests and Docker.`

## Suggested LinkedIn / CV bullet

Built a Python simulator for 5G network slicing that compares latency-based, K-Means and random user allocation strategies, calculates slice load/utilization metrics, detects overloads and generates reproducible analytical charts.
